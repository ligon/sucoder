* Overview
Welcome to the Org authoring playground. This file demonstrates the workspace
skills we’ve been codifying: document structure, lists, markup, TODO workflows,
hyperlinks, tables, capture-style entries, source blocks, citations, exporting, and
more. Use it as a reference or clone its patterns when drafting new documentation.

* Structure & Drawers
:PROPERTIES:
:Owner: Assistant
:END:
:LOGBOOK:
- State "TODO"       from              <2025-11-09 Sun>
:END:

The property drawer tracks metadata; =LOGBOOK= stores state changes. Folding a subtree
places all drawers upfront so the body stays tidy. Below we demonstrate headlines,
refiling, and linking.

* Source Blocks & Results

#+begin_src python

    print(f"Iteration {i}")

#+RESULTS: tiny-python-demo

Source blocks use lowercase directives, explicit =:results=, and avoid auto-eval during export.
